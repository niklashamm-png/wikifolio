## Key Principles

1. **All content in English.** The wikifolio and all user-facing materials must be in English.
2. **No return guarantees.** Never imply guaranteed returns or certainty about future performance.
3. **Always include a disclaimer.** Every public post, article, or newsletter must contain a risk note.
4. **Transparency first.** Document every rebalancing decision with reasoning.
5. **Rule-based framing.** Emphasise the systematic nature – no gut feeling, no emotion.
6. **Professional tone.** The audience is sophisticated and return-focused. Avoid hype language.
